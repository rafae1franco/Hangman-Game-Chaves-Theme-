# 🎮 Hangman Game (Chaves Theme)

A simple Hangman game developed in Python, inspired by the classic TV show *Chaves*.  
The player must guess the word letter by letter or try to guess the full word.

---

## 🚀 Features

- Random word selection
- Hint system for each word
- Letter-by-letter guessing
- Full word guessing option
- Score system
- Attempt control based on difficulty
- Tracks used letters
- Modular structure (separated data file)

---

## 🧠 How It Works

1. The game selects a random word.
2. A hint is displayed to help the player.
3. The player can:
   - Type a single letter
   - Try to guess the entire word
4. The game updates the progress and score.
5. The game ends when:
   - The player guesses the word
   - The attempts run out
